Manual de usuario

Manual donde se especifica la funcion de cada una de las funciones de doorway

1.-
DoorWay
Es el nombre de nuestro Software. Es un software de gestión de inventario, se encarga de facturar, cualquier tipo de compra, de guardar los datos de los productos de los cliente y generar reportes en PDF, de las ventas por días, mensual, o incluso anual
2.0.-
Cliente
Es el usuario que comprara artículos, es decir es el que seleccionara los productos a adquirir, se procederá a crear una lista de compras con los artículos que el cliente escogió, luego, se realizara el pago por cada uno de ellos
2.1.-
Agregar
Esta opción tiene como función agregar nuevos clientes al Sistema.
2.2.-
Lista de Clientes
Es una lista de todos los clientes guardados en la base de datos, aqui se pueden editar y eliminar dichos clientes
3.0.-
Facturación
La Facturación es donde están los productos que el cliente adquirió o compro, el monto pagado y la fecha, hora de cuando se efectuó la compra
3.1.-
Nuevo
Esta opción permite crear una nueva factura.
3.2.-
Anular
Al hacer clic en esta opción se anulara la factura elegida y los productos que el cliente haya escogido.
4.0.-
Reporte
Reporte dde todas las facturas producidas
4.1.-
Reporte
En el reporte se pueden ver las ventas hechas ya sea en el Mes, o en el Día, o incluso de forma Anual, este reporte será generado en un archivo PDF.
5.0.-
Inventario
En el Inventario podemos ver la cantidad de artículos disponibles, así como el Nombre del Artículo, Cantidades disponibles para la venta, y el Precio de Cada uno de ellos.
5.1.-
Añadir Producto
Opción que permite añadir un nuevo producto al Inventario y la Base de Datos, aquí se necesitarían introducir el precio del producto y sus cantidades disponibles.
5.2.-
Lista de Productos
Es una lista de todos los productos guardados en el inventario, aqui se pueden editar y eliminar dichos productos
5.3.-
Crear Orden de Compra
Opción que permite crear una nueva orden de compra para compras productos y añadirlos al inventario
5.4.-
Manejar Órdenes de Compra
Consiste en manejar las ordenes de compras ya creadas, al efectuarse la compra se le añadirá la cantidad del producto.
6.0.-
Ayuda
Ofrece una ayuda para el cajero, trabajador de cómo usar este Software, también incluye la función de Acerca de, Manual de usuario
6.1.-
Manual de usuario
En esta vista estan explicadas todas las secciones de la aplicacion
6.2.-
Acerca de..
Se pueden ver la versión del Software, y el Nombre de los Creadores así como el Nombre del Software.
